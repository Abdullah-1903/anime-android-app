# anime-android-app

Android-applikasjon utviklet med Kotlin og Jetpack Compose.

## Teknologier
- Kotlin
- Jetpack Compose
- MVVM
- Retrofit
- Room Database
- Navigation Compose
- Coil

## Funksjoner
- Hente anime-data fra Jikan API
- Søke etter anime via ID
- Lagre egne anime-ideer lokalt
- Redigere og slette lagrede ideer
- Matchup-minispill basert på anime-popularitet
